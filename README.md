# 📚 StudyCafe Frontend

`StudyCafe` 백엔드(Spring Boot + Kotlin)와 연동되는 React 웹 프론트엔드입니다.

## 빠른 시작

```bash
npm install
npm run dev
# → http://localhost:5173
```

백엔드가 8080 포트에서 실행 중이어야 합니다. `.env` 참고.
